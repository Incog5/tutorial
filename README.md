# ลิขสิทธิ์
ข้อความหรือรูปภาพที่พิมพ์หรือวาดขึ้นใหม่ที่ปรากฏในเว็บไซต์นี้ กำหนดให้เป็นสาธารณสมบัติ (public domain) ข้อกำหนดด้านสัญญาอนุญาตบันทึกใน UNLICENSE ใจความสำคัญคือผู้อ่านนำเนื้อหานี้ไปใช้โดยไม่มีเงื่อนไข
